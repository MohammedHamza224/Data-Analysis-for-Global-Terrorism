📊 Terrorism Data Analysis Project (1970–2018)

This project analyzes global terrorism incidents from 1970 to 2018 using Python and data visualization techniques.
The objective is to identify trends, patterns, and insights related to terrorist activities across different countries,
regions, attack types, and weapon categories.
________________________________________
🔍 Key Objectives

Analyze global terrorism trends over time.
Identify countries and regions most affected by terrorism.
Explore attack and weapon type distributions.
Examine the success rates of terrorist incidents.
Create insightful visualizations for data storytelling.
________________________________________

🛠️ Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Plotly

VS Code Notebook
________________________________________

📈 Key Findings

•	 Iraq, Afghanistan, Pakistan, and India experienced the highest number of incidents. 
•	Terrorist activities are concentrated in the MENA and South Asian regions. 
•	Significant growth in incidents occurred after the early 2000s. 
•	Attack success rates vary across attack and weapon categories. 
•	Terrorism trends show notable fluctuations across different decades.
